# @repo/types

Shared TypeScript types used across the monorepo.

## Modules

| Import | Exports |
|--------|---------|
| `@repo/types` | All types (barrel) |
| `@repo/types/api` | ApiResponse, ApiError, ApiSuccess |
| `@repo/types/pagination` | PaginationParams, PaginatedResult |
| `@repo/types/user` | User, CreateUserInput |
| `@repo/types/role` | Role, hasMinimumRole |
| `@repo/types/theme` | Theme, ResolvedTheme |
