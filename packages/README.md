# Packages

Shared libraries used across apps in this monorepo.

## Packages

| Package | Description |
|---------|-------------|
| [`@repo/ui`](./ui) | Shared UI components and design system |
| [`@repo/config`](./config) | Shared ESLint, TypeScript, Prettier configs |
| [`@repo/core`](./core) | Backend infrastructure (env, logger, errors, validation) |
| [`@repo/types`](./types) | Shared TypeScript types |
| [`@repo/db`](./db) | Database platform (hexagonal adapters) |
| [`features/`](./features) | Domain features (Sprint 4+, placeholder) |
